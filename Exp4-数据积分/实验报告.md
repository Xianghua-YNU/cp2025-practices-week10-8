## 二、实验原理

### 2.1 梯形积分法

对于离散的速度数据点$(t_0,v_0),(t_1,v_1),...,(t_N,v_N)$，总距离（位移）可通过梯形法则计算：

$$
\text{距离} \approx \sum_{i=0}^{N-1} \frac{v_i + v_{i+1}}{2} (t_{i+1} - t_i)
$$

### 2.2 累积距离计算

使用`scipy.integrate.cumulative_trapezoid`计算每个时间点的累积距离：

$$
\text{累积距离}(t_i) = \int_{t_0}^{t_i} v(t)dt
$$

import numpy as np
from scipy.integrate import cumulative_trapezoid
import matplotlib.pyplot as plt


def main():
    try:
        # 读取数据
        data = np.loadtxt('Velocities.txt')
        t = data[:, 0]  # 时间列
        v = data[:, 1]  # 速度列

        # 计算总距离
        total_distance = np.trapz(v, t)
        print(f"总运行距离: {total_distance:.2f} 米")

        # 计算累积距离
        distance = cumulative_trapezoid(v, t, initial=0)

        # 绘制图表
        plt.figure(figsize=(10, 6))

        # 绘制速度曲线
        plt.plot(t, v, 'b-', label='速度 (米/秒)')

        # 绘制距离曲线
        plt.plot(t, distance, 'r--', label='距离 (米)')

        # 添加标题和标签
        plt.title('速度与距离随时间变化')
        plt.xlabel('时间 (秒)')
        plt.ylabel('速度 (米/秒) / 距离 (米)')

        # 添加图例
        plt.legend()

        # 显示网格
        plt.grid(True)

        # 显示图表
        plt.show()
    except FileNotFoundError:
        print("错误: 未找到 Velocities.txt 文件。")


if __name__ == "__main__":
    main()

### 4.2 误差分析
分析梯形积分法计算距离的误差来源
离散化误差：梯形积分法是基于离散的数据点进行计算的。实际的速度 - 时间函数是连续的，但在数据采集或数值计算中，只能得到有限个离散的时间点及其对应的速度值。这些离散点之间的函数信息被近似为梯形，忽略了函数在离散点之间的真实变化，从而产生误差。离散点越稀疏，这种误差就越大；反之，增加离散点的数量可以减小离散化误差。
函数近似误差：梯形积分法假设在每个时间间隔内，速度函数是线性变化的，即用梯形来近似速度 - 时间曲线下的面积。然而，实际的速度函数可能并非严格线性，在时间间隔内可能存在非线性变化。当速度函数的非线性程度较高时，使用梯形近似会导致较大的误差。
数据测量误差：如果用于计算的时间和速度数据本身存在测量误差，那么这些误差会传递到梯形积分的计算结果中。例如，测量时间的仪器精度有限，或者测量速度时存在一定的误差范围，都可能导致最终计算出的距离产生偏差。即使测量误差是随机的，在多次测量取平均后可能会有所减小，但仍然会对积分结果产生一定的影响。
边界误差：在计算累积距离时，梯形积分法从初始时间开始逐步计算每个时间间隔的面积并累加。在积分的起始和结束边界处，由于只有单侧的梯形，与内部的梯形相比，其面积计算可能会存在一定的偏差。特别是当起始或结束点的速度与相邻点的速度差异较大时，边界处的梯形近似会引入相对较大的误差。
