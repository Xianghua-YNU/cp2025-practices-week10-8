## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（$N$ 必须为偶数），并给出公式：
$$
\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]
$$

### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
$$
\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]
$$

import numpy as np

def f(x):
    # TODO: 实现被积函数 f(x) = x^4 - 2x + 1
    return x ** 4 - 2 * x + 1

def trapezoidal(f, a, b, N):
    """
    梯形法数值积分
    :param f: 被积函数
    :param a: 积分下限
    :param b: 积分上限  
    :param N: 子区间数
    :return: 积分近似值
    """
    # TODO: 实现梯形法则积分
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    y = f(x)
    integral = 0.5 * h * (y[0] + y[-1] + 2 * np.sum(y[1:-1]))
    return integral

def simpson(f, a, b, N):
    """
    Simpson法数值积分
    :param f: 被积函数
    :param a: 积分下限
    :param b: 积分上限
    :param N: 子区间数（必须为偶数）
    :return: 积分近似值
    """
    # TODO: 实现Simpson法则积分
    # 注意：需先检查N是否为偶数
    if N % 2 != 0:
        raise ValueError("Simpson 法则要求 N 必须为偶数")
    h = (b - a) / N
    x = np.linspace(a, b, N + 1)
    y = f(x)
    # 奇数索引（1,3,5,...,N-1）
    odd_sum = np.sum(y[1:N:2])
    # 偶数索引（2,4,6,...,N-2）
    even_sum = np.sum(y[2:N:2])
    integral = h / 3 * (y[0] + 4 * odd_sum + 2 * even_sum + y[N])
    return integral


def main():
    a, b = 0, 2  # 积分区间
    exact_integral = 4.4  # 精确解

    for N in [100, 1000]:  # 不同子区间数
        # 调用积分函数并计算误差
        trapezoidal_result = trapezoidal(f, a, b, N)
        simpson_result = simpson(f, a, b, N)
        
        # 计算相对误差
        trapezoidal_error = abs((trapezoidal_result - exact_integral) / exact_integral)
        simpson_error = abs((simpson_result - exact_integral) / exact_integral)

        # 输出结果（模板已给出）
        print(f"N = {N}")
        print(f"梯形法则结果: {trapezoidal_result:.8f}, 相对误差: {trapezoidal_error:.2e}")
        print(f"Simpson法则结果: {simpson_result:.8f}, 相对误差: {simpson_error:.2e}")
        print("-" * 40)

if __name__ == '__main__':
    main()

### 4.2 误差分析
1. 不同N下的误差表现
 
- 当N = 100时，梯形法则相对误差为0.009091 ，Simpson法则相对误差为0 （示例数据）。随着N增大到1000 ，梯形法则相对误差减小到0.000091 ，Simpson法则相对误差仍为0 （示例数据）。可见，N增大，两种方法误差都减小，Simpson法则误差更小且趋近于0更快。
 
2. Simpson法则与梯形法则误差的数量级差异及原因
 
- 梯形法则基于线性近似，其误差阶为O(h^2) （h为子区间宽度，h=\frac{b - a}{N} ）；Simpson法则基于二次近似，误差阶为O(h^4) 。随着N增大，h减小，h^4 比h^2 减小速度快得多，所以Simpson法则误差数量级比梯形法则小，精度更高。
 
3. N增大时误差的变化趋势
 
- 对于梯形法则和Simpson法则，N增大意味着子区间宽度h减小。从误差公式角度，梯形法则误差与h^2 相关，Simpson法则误差与h^4 相关，h减小时，两者误差都减小，且Simpson法则误差减小速度远快于梯形法则。

## 五、结论
1. 两种方法的适用场景和优劣
 
- 梯形法则：
 
- 优点：原理简单，实现容易，计算量相对小，在对精度要求不高、函数变化较平缓、计算资源有限或追求快速计算的场景适用。
 
- 缺点：精度低，对于复杂函数或高精度要求情况，需大量子区间，计算量剧增。
 
- Simpson法则：
 
- 优点：精度高，对光滑函数收敛快，在高精度计算、函数较光滑场景优势明显。
 
- 缺点：要求子区间数为偶数，限制使用灵活性；对非光滑函数效果可能不好。
 
2. 实际工程中选择数值积分方法
 
- 若工程计算对精度要求低、函数简单、计算资源受限或需快速估算，选梯形法则，如一些初步设计阶段的粗略计算。若对精度要求高、函数光滑且计算资源允许，选Simpson法则，如精密物理模拟、高精度工程测量数据处理等场景。
